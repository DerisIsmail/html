ATRIBUT PADA TABLE DI HTML

### 3 atribut ini bisa langsung di html atau bisa di css
1. border = untuk membuat border / garis 
2. cellspacing = untuk menambah jarak antar cell
3. cellpadding = untuk memberi luas pada cell

###
4. colspan = untuk menggabungkan 2 buah cell / lebih pada kolom (col)
5. rowspan = untuk menggabungkan 2 buah cell / lebih pada baris (row)
